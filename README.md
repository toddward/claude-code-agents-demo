# Designing Custom Agents Demo

A hands-on demo for the DevX Working Group showing how to design and use custom agent personas in Claude Code. There is an associated slide deck (`designing-custom-agents.html`), but the demo stands alone as a self-contained walkthrough.

## What You'll See

- Define specialized agent personas directly in `CLAUDE.md`
- Build a simple Node.js/Express task management API using spec-driven development (`TASKS.md`)
- Invoke custom agents to **review**, **secure**, **document**, and **test** the application — all without leaving Claude Code
- Watch how agent-driven prompting produces more targeted, actionable feedback compared to generic prompts

## Prerequisites

- [Claude Code](https://claude.ai/code) installed and authenticated
- Node.js 18+ installed
- Familiarity with basic terminal commands

Follow the step-by-step instructions in [`Demo_Directions.md`](Demo_Directions.md).
